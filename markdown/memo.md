## Python

### class

#### `__new__()` と `__init__()`

コンストラクタとして、`__new__()`, `__init__()` の順で呼ばれ、前者はインスタンスオブジェクトを生成、後者はそれを初期化している。後者の返り値が `None` であるのはそのため。

なお、全てのクラスは `object` のサブクラスらしい。
